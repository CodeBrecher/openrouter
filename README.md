# OpenRouter Build 🚀

A monorepo project integrating the OpenRouter API, built following Harkirat's tutorial. 

## Tech Stack
* **Framework:** Next.js
* **Monorepo:** Turborepo
* **AI API:** OpenRouter
* **Package Manager:** Bun

## Getting Started

1. Clone the repository:
   `git clone https://github.com/CodeBrecher/openrouter.git`
2. Install dependencies (using Bun, as configured):
   `bun install`
3. Set up environment variables:
   Create a `.env` file in your app directory and add your `OPENROUTER_API_KEY`.
4. Start the development server:
   `bun run dev`
